# Supermarket Sales Analysis

## Project Overview

This project analyzes supermarket sales transactions from January 2019 to March 2019 to identify revenue drivers, customer purchasing behavior, product performance, branch performance, and sales trends.

The analysis was conducted using SQL, Python, and Power BI to demonstrate end-to-end data analytics skills, from data cleaning and exploration to dashboard development and business recommendations.

---

## Business Problem

Management wants to better understand:

* Which products generate the highest revenue?
* Which customer segments contribute the most sales?
* Which branches perform best?
* What payment methods are preferred?
* How do sales vary over time?
* What actions can improve business performance?

---

## Dataset Information

* Dataset: Supermarket Sales Dataset
* Number of Transactions: 1,000
* Time Period: January 2019 – March 2019
* Branches: Giza, Cairo, Alex
* Analysis Tools:

  * SQL (MySQL)
  * Python (Pandas, Matplotlib)
  * Power BI

## Key Findings

1. Total sales reached approximately 322,967.

2. Food & Beverages generated the highest revenue and achieved the highest customer ratings.

3. Female customers generated higher sales than male customers.

4. Cash was the most frequently used payment method, although payment method usage was relatively balanced.

5. Saturday generated the highest sales revenue.
